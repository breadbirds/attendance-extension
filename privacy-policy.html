부울경 출석처리 자동화 - 개인정보처리방침

최종 수정일: 2024년 9월 10일

1. 수집하는 정보

이 확장프로그램은 개인정보를 수집하지 않습니다. 다음 설정 정보만 사용자의 브라우저에 로컬 저장됩니다:

  • 반 번호 (1반/2반)
  • Google Sheets 출석부 ID
  • Google Apps Script 웹앱 URL
  • 초기화 완료 상태

2. 정보 사용 목적

저장된 설정 정보는 출석 관리 자동화 기능 제공 목적으로만 사용됩니다.

3. 정보 저장 및 보관

모든 데이터는 사용자의 브라우저 로컬 스토리지에만 저장되며, 외부 서버로 전송되지 않습니다. 확장프로그램을 제거하면 모든 데이터가 함께 삭제됩니다.

4. 제3자 제공

수집된 정보는 어떠한 제3자에게도 제공되지 않습니다.

5. 보안

모든 통신은 HTTPS를 통해 암호화되며, Google Apps Script는 Google의 보안 정책에 따라 보호됩니다.

6. 문의

개인정보처리방침에 대한 문의사항이 있으시면 GitHub Issues를 통해 연락해주세요.
